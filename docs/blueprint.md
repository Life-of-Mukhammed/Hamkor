# **App Name**: I-HAMKOR

## Core Features:

- Uzbek (Cyrillic) UI Support: Implement full localization for the user interface, ensuring all text and labels are displayed in the Uzbek (Cyrillic) script.
- Dynamic Dashboard Ad Carousel: A top-level banner carousel on the dashboard showcasing premium brands like UzAuto Motors, Artel, and Akfa with HD images and animations, rotating every 3-5 seconds.
- SPA Navigation System: A sidebar-driven Single Page Application navigation system supporting 24 distinct, functional sections with fluid transitions between them.
- Live Auction Display & Bidding UI: User interface to display dynamic price bidding, updated every 3 seconds, including a clear view of bid history for active auctions (using mocked data for MVP).
- GPS Logistics Map & Simulation: An integrated map interface displaying simulated moving truck icons and a real-time ETA calculator (using mocked GPS data for MVP).
- AI Risk Score Tool & Blacklist Check Display: A user interface component for INN search, displaying a hypothetical AI-generated risk score derived from a simulated 'Blacklist' database, for security analysis.
- PDF Preview & Contract Workflow UI: Functional simulation of PDF preview for electronic signatures and the user interface for a multi-step contract approval workflow (without actual ERI backend integration for MVP).

## Style Guidelines:

- Color Scheme: Light/White Premium, emphasizing professionalism and clarity.
- Primary color: A vibrant business blue (#1A5AE0), signifying trust and stability.
- Background color: A very light, desaturated blue-grey (#EEF2FA) to maintain a clean, premium feel.
- Accent color: A warm, rich golden-orange (#E0AA1A) to highlight interactive elements and notifications.
- Body and headline font: 'Inter' (sans-serif), chosen for its modern, clean lines, and high legibility suitable for a professional B2B platform.
- Professional, outline-style B2B icons or premium text labels, avoiding informal graphics or emojis.
- User interface elements like buttons, tables, and cards will feature a minimalist, 'Premium' aesthetic, incorporating subtle Glassmorphism or Soft UI effects.
- Subtle micro-animations such as ripple effects on clicks, hover glows on interactive elements, and fluid transitions between SPA sections, enhancing the premium feel.
